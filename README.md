## Exercício `IMC`

Calcule o índice de massa corporal de uma pessoa, dados altura (em m) e peso (em kg).

IMC = peso / altura²

```
Altura (m)..: 2
Peso (kg)...: 80

IMC: 20,0 kg/m²
```
